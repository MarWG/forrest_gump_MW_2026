# Proyecto Forrest Gump - Primer Parcial Programación Web 1

## Descripción del Proyecto
Este es un sitio web simple, perteneciente a un proyecto evolutivo dedicado a la película "Forrest Gump" (1994), desarrollado como parte del primer parcial de Programación Web 1. El proyecto cumple con todos los requisitos establecidos en la consigna.

## Estructura del Proyecto

ForrestGump-Proyecto/
├── index.html          # Archivo principal HTML
├── styles.css         # Hoja de estilos CSS
├── imagenes/           # Carpeta para imágenes
└── README.md           # Este archivo


## Características Implementadas

### Estructura HTML Semántica
- Uso correcto de etiquetas HTML5: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, git `<aside>`, `<footer>`
- Estructura ordenada siguiendo los requerimientos de la consigna
- Navegación funcional con enlaces internos

### Secciones Implementadas
1. **Inicio (Home)**: Introducción, género, imagen promocional
2. **Sobre la Película**: Sinopsis, contexto, análisis personal, locaciones
3. **Reparto**: Actores principales con descripciones detalladas
4. **Producción**: Estudios, proceso de filmación, efectos especiales
5. **Curiosidades y Premios**: Anécdotas, premios Oscar, impacto cultural
6. **Contacto**: Estructura para formulario futuro, redes sociales

### Diseño CSS
- Estilos para texto y colores.
- Diseño responsivo básico
- Efectos visuales sutiles (hover, gradientes, sombras)
- Paleta de colores profesional
- Tipografía legible y apropiada

## Cumplimiento de Requisitos

### Técnicos
- Solo HTML y CSS básico (sin frameworks, bibliotecas externas o JavaScript)
- Estructura semántica correcta
- Código organizado y prolijo

### Contenido
- Contenido original y creativo sobre Forrest Gump
- Análisis personal en la sección "Sobre la Película"
- Información detallada y bien investigada

### Estructura
- Orden correcto de contenidos según la consigna
- Etiquetas HTML apropiadas para cada sección
